### ▶ Tech Stacks

- **Front-end** : HTML, CSS, Thymeleaf, JQuery <br>
- **Back-end** : Java, SpringBoot, JPA, Hibernate <br>
- **build** : gradle  <br>
- **DB** : MySQL  <br>
- **deployment** : Docker, AWS  <br>
- **Tools** : IntelliJ, Postman   <br>

### ▶ 사용자 Flow
- `회원등록` `회원수정` `회원삭제`
- `상품조회` `상품 상세조회`
- `장바구니 담기` `장바구니 조회` `장바구니 수량 수정` `장바구니 삭제`
- `상품 주문 및 결제`
